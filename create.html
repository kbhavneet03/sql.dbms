CREATE COMMAND

CREATE command is a DDL command.DDL stands for Data Definition Language. The DDL commands are used to define or alter the structure of the table. Some common examples of DDL commands are create table command, alter table command, etc.The create table command is the DDL command for creating a table in a MySQL database.

Syntax:
CREATE TABLE tableName(ColumnName datatype size constraint);

Each column in a table must have a name, data type, and size (size defines the maximum length of data that a given column may carry). CREATE TABLE is the keyword telling the database system what you want to do. In this case, you want to create a new table. The unique name or identifier for the table follows the CREATE TABLE statement. Then in brackets comes the list defining each column in the table and what sort of data type it is.

Create a Table using another table

A copy of an existing table can be created using a combination of the CREATE TABLE statement and the SELECT statement. The new table has the same column definitions. All columns or specific columns can be selected. When you will create a new table using the existing table, the new table would be populated using the existing values in the old table.

Syntax:
CREATE TABLE NEW_TABLE_NAME AS
SELECT [ column1, column2...columnN ]
FROM EXISTING_TABLE_NAME
[ WHERE ]

 Here, column1, column2... are the fields of the existing table and the same would be used to create fields of the new table.

SELECT Command

In SQL ( or Structured Query Language ), the SELECT statement is the most extensively used command. It's used to look up information in one or more database tables and views and obtain the data that matches the criteria we specify.The SELECT query in SQL is used in fetching data from a database table. The fetching can be adjusted to retrieve data that meets a set of criteria. The data is returned in a tabular format, where the table is called the result set.
The SELECT statement in SQL is frequently used alongside the FROM, WHERE, GROUP BY, LIMIT, etc. clauses. The SELECT statement in SQL is used to retrieve data from a database table, filtering and working with data types. The accessed data is in a tabular format. This result table is called the result set. The SELECT clause in SQL is usually used in conjunction with the FROM clause to define the table name(s) and the WHERE and ORDER BY clauses to filter or sort the data based on the specified condition(s).

Syntax of SELECT Statement in SQL The basic syntax for using SELECT is to specify the column names we want to retrieve, followed by the FROM clause to designate the name of the database table:
SELECT
column1,
column2,
……
FROM table_name;

 In case we want to retrieve all the columns, we can replace writing individual column names with *.

SELECT * FROM table_name;
